HTML

HTML, which stands for HyperText Markup Language, is the standard language used to create and design web pages. It provides the basic structure and content of a webpage, which is then styled and enhanced by other technologies like CSS (Cascading Style Sheets) and JavaScript.

More about HTML

CSS

CSS, which stands for Cascading Style Sheets, is a stylesheet language used to describe the presentation and styling of a document written in HTML (or XML). While HTML defines the structure and content of a webpage, CSS controls how that content looks, including layout, colors, fonts, and more.

More about CSS

JS

JS, or JavaScript, is a versatile and widely-used programming language primarily known for adding interactivity and dynamic behavior to web pages. It is one of the core technologies of the web, alongside HTML (for structure) and CSS (for styling). JavaScript enables developers to create rich, interactive, and responsive user experiences.
